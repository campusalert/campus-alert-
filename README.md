# 📌 Project Overview

This project helps students quickly send emergency alerts to the campus security team.

The security panel receives alerts instantly using Socket.IO.  
Security staff can:

- Approve alerts
- Dismiss alerts
- Send emergency SMS notifications
- Monitor live reports
- View alert history
- Track priority levels

The system works in real-time and improves campus safety response.

# ✨ Features

## 👨‍🎓 Student Panel
- Send emergency reports
- Enter student name
- Select emergency type
- Select priority level
- Real-time report submission
- Responsive UI

## 🛡️ Security Panel
- Live emergency monitoring
- Approve/Dismiss reports
- Live digital clock
- Alert history section
- Emergency statistics dashboard
- Send SMS alerts directly
- Responsive modern interface

## 📱 SMS Integration
- Twilio SMS API integration
- Instant alert notifications to registered numbers

## 🌐 Deployment
- Hosted on Render
- GitHub integrated project
